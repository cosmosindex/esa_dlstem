# SV248S — Resolution Statistics

Dataset path: `/data/ESA_DLSTEM_2025/data/trafic/SV248S`

Source: Jilin-1 satellite video. 248 targets across 6 videos, 4 classes. Frames are TIFF format.

## Dataset Overview

- **Satellite:** Jilin-1, nadir GSD = 0.92 m/pixel
- **Original frame size:** 4096 x 2160 (some videos 3840 x 2160); sequences are crops from these full frames
- **Frame rate:** 10 fps (native), interpolated to 25 fps via VFI
- **Task:** SOT only — each sequence tracks a single target. Not designed for MOT or detection benchmarks.
- **Categories:** car, car-large, plane, ship (4 classes)
- **Annotations:**
  - `.rect` — per-frame upright bounding box (x, y, w, h)
  - `.poly` — per-frame tight polygon (4/5/6/8/dense points), close to mask-level precision. Both BBox-based and mask-based trackers can be evaluated.
  - `.state` — per-frame visibility flag (0=normal, 1=invisible, 2=occluded)
  - `.attr` — 10 sequence-level attributes (CSV integers)
  - `.abs` — JSON metadata (class, difficulty level, sequence length, etc.)
- **Note:** The polygon annotation is per-instance contour, not dense semantic segmentation (e.g. DAVIS-style). Only the tracked target is annotated per frame.
